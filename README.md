# 🤖 NSP Chatbot

NSP Chatbot is a modern AI-powered chatbot project built using Python Flask, HTML, CSS, JavaScript, MySQL, and Groq AI API.

The chatbot supports:

- AI text chat
- Voice input and voice response
- File upload
- Chat history
- Code generation
- Smart spelling correction
- Modern UI design

---

# 🚀 Features

## ✅ AI Chat Support
User can ask any question and get AI-generated responses.

## ✅ Voice Assistant
- User can speak using microphone
- Chatbot converts voice to text
- Bot can reply using voice

## ✅ File Upload
User can upload:
- Images
- Documents
- Attachments

## ✅ Search History
- Saves all chats in MySQL
- View previous searches
- Delete single history
- Delete all history

## ✅ Coding Assistant
- Generates clean code
- Supports Python, Java, C, C++, HTML, CSS, JS, etc.
- Gives formatted code blocks

## ✅ Smart AI Understanding
- Understands spelling mistakes
- Understands shortcut words
- Gives human-like responses
